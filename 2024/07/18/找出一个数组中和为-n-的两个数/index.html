找出一个数组中和为 n 的两个数

找出一个数组中和为 n 的两个数

问题描述

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出和为目标值 target 的那两个整数，并返回它们的值。

方法一 暴力法

暴力法的思路是遍历数组中的所有可能的两数组合，并检查它们的和是否等于目标值 n。这种方法时间复杂度较高，但易于理解和实现。

• 时间复杂度：O(n^2)
• 空间复杂度：O(1)

方法二 双指针法

双指针法要求数组是已排序的。如果数组未排序，需要先进行排序。然后使用两个指针分别指向数组的头和尾，根据它们的和来调整指针的位置，直到找到目标值 n。

• 时间复杂度：O(n log n)（排序） + O(n)（双指针遍历） = O(n log n)
• 空间复杂度：O(1)

方法三 哈希表法

哈希表法通过遍历数组并将每个元素存入哈希表，同时检查哈希表中是否存在与当前元素和为目标值 n 的另一个元素。如果存在，直接返回这两个元素。

• 时间复杂度：O(n)
• 空间复杂度：O(n)
